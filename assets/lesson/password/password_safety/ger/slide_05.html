Kaspersky also checks if your password has been leaked in a database breach.

Database breaches can happen when a website or service is hacked or otherwise attacked. Unfortunately, your data can be leaked in the process. Sometimes, this data is released into the public.


If your password was leaked, it’s a good idea to create a new password as soon as possible.
