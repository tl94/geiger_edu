Kaspersky überprüft auch, ob dein Passwort in einem Leak vorkommt.

Leaks passieren wenn eine Webseite oder ein Dienst gehackt werden und auf ihre Daten zugegriffen wird. So kann es sein, dass deine Daten, insbesondere E-Mails und Passwörter öffentlich freigegeben werden.


Wurde dein Passwort geleaket, solltest du so schnell wie möglich ein neues erstellen.
